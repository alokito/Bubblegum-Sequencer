Bubblegum Sequencer

=== DEPENDENCIES ===
At this level, you will need to add the following libraries:

- ij.jar (ImageJ; http://rsbweb.nih.gov/ij/download.html)

For Mac users:
- mmj.jar (3rd party MIDI support; http://www.humatic.de/htools/mmj.htm)

If you want to use an Arduino for tempo input or blinking LEDs:
- rxtx-2.1.7.jar
